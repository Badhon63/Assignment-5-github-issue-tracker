1️⃣ What is the difference between var, let, and const?
var is nonscoped, meaning it can be called outside functions, codeblocks, etc without any restrictions.
let variable values can be changed directly or indirectly.
const variable values can not be changed directly.


2️⃣ What is the spread operator (...)?
spread operator is used to copy arrays and objects into other arrays and objects easily.


3️⃣ What is the difference between map(), filter(), and forEach()?
map() makes a new array with changed values.
filter() makes a new array with only the items that pass a test.
forEach() loops through each item of the array.


4️⃣ What is an arrow function?
arrow function is a shorter way to write a function (=>) instead of the function keyword.


5️⃣ What are template literals?
template literals are backtiks that is used to directly apply html in js code. js variables and functions can also be easily called in the template literals.
